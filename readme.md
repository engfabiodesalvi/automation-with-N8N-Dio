# Santander 2025 - Automação com N8N

Compartilho minha nova conquista.

O bootcamp Santander 2025 - Automação com N8N abordou os conceitos básicos sobre a automação de processos utilizando o N8N.

Durante a trilha aprendemos como realizar a instalação do ambiente de automação utilizando o Node.js e Docker local e uso em ambiente Cloud.

Foram implementadas automações básicas que exemplificaram a forma como os dados são recebidos, processados e enviados utilizando os respectivos nós no ambiente!

Acesso a dados de planilhas do google drive, Telegram e WhatsApp.

Envio de dados para e-mail, Telegram e WhatsApp.

Foi demonstrado formas de proteção de acesso à automação utilizando token e autenticação.

Foi uma excelente oportunidade para aprofundar meus conhecimentos na área de automação de processos utilizando Inteligência Artificial (AI) e N8N!
