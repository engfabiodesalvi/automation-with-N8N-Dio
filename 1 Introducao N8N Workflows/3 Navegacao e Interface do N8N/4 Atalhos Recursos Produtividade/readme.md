# Navegação e Interface do N8N
Atalhos e Recursos de Produtividade no N8N.

## Hands-On
Identificar e utilizar atalhos do teclado para otimizar e aumentar a produtividade das suas tarefas diárias.

* **Zoom:** Botões mais(`+`) e menos(`-`). Tecla 1 volta para a escala normal.
* **Toolbox Nodes Panel:** `Tab`. As teclas `seta para cima` e `seta para baixo` podem ser utilizadas para navegar neste menu. A tecla `Enter` seleciona o item desejado.
* **Fechar uma janela de cnfiguração:** Tecla `esc`.
* **Navegar entre os nós:** Setas `esquerda` e `direita`.
* **Configurar um nó:** Tecla `Enter`.
* **Executar o workflow:** Teclas `Crtl + Enter`.
* **Salvar o workflow** Teclas `Crtl + S`.
* **Adicionar uma nota** Teclas `Shift + S`.
* **Deletar um nó:** Tecle `Del` quando o nó estiver selecionado.
* **Mostrar o console:** Prescionar a tecla `L`.
