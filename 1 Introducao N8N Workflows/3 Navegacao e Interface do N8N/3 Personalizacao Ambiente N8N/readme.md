# Navegação e interface do N8N

Personalização do ambiente no N8N.

## Workspace

Ambiente de trabalho, espaço de trabalho.

* **Workflows**
* **Projects/Folder**
* **Credencials**

Menus que alteram ou afetam diretamente a **Workspace**

* **Customs**
* **Settings**

## Hands-On

Será explorado o **Panel** dos itens **Admin Panel** e **(User login)** do **Side bar**

### Admin Panel

* **Dashboard**
    * Mostra a *workspace currently online*
    * Contem os detalhes do plano da assinatura do usuário.
    * Contém as *executions* e *active workflows*
    * Opção de *upgrade* do plano
    * Mostra a versão do **N8N** em execução

* **Manage**
    * **Workspace**
        * Mostra o plano corrente
        * Mostra a versão corrente do **n8n**.
        * É possível mdificar a versão crrente do **n8n**
        * Configurar o **Time zone**
        * Habilitar/desabilitar **Verified community nodes**
        * Habilitar/Desabilitar **Disable all community nodes**

    * **Billing**
        * **Payments**
            Informações de pagamento
        
    * **Export**
        * **Export Data**
            Realiza o download de uma cópia do **workflow** do último backup do **N8N** realizado na nuvem (*cloud*)

    * **Help Center**
        Para ajuda com o **N8N**
        * **Get started**
        * **Creatting a workflow**
        * **Connecting to apps**
            Tópico mais acessado por conter ajuda para conectar com aplicativos de terceiros.            
        * **Submit support ticket**
        * **Frequently asked questions**


* **Settings**
    * **Personal**
        * Dados pessoais **Personal Settings**

        * **Security**
            Modificar o **Password** e habilitar  **Two-factor authentication (2FA)**
        * Personalização do tema

    * **Users**
        * Convidar outros usuários para participarem do projeto
    
    * **n8n API**
        Criar apis utilizando o n8n

    * **External Secrets**
        Camada de segurança extra para manipular dados sensíveis

    * **Environment**
        Integrar com ambiente de desenvolvimento.
    
    * **SSO**
        Sistema unificado de login.

    * **LDAP**
        Protocolo de comunicação e autenticação para ambientes integrados.
    
    * **Log Streaming**
        Log detalhado (versão *Enterprise*)
