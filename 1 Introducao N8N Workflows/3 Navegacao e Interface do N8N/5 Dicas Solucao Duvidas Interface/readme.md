# Navegação e Interface d N8N
Dicas e Solução de Dúvidas Sobre a Interface do N8N.

* Sempre Verificar a versão do **n8n** que está em uso.
* Estar atento entre as versões **Lasted Beta** e **Lasted Stable**. As versões beta podem não ser estáveis em certos projetos.
* Sempre verificar os **Changelogs** das versões para verificar as notas de modificações e avisos.

## Hands-On
Identificar e resolver dúvidas comuns por conta própria, otimizando seu uso do sistem.

* Acessar **Visit foruns** detro da aba **Admin panel**.
* Explorar todo o ecossistema do N8N.
