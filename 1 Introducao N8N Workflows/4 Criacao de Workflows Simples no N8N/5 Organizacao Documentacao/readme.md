# Criação de Workflows Simples no N8N
Organizando e Documentando Workflows no N8N.

Adotar boas práticas de organização:
* Nomeie sempre o step (passo) corretamente,
* Use o botão **Tidy Up** para deixar a automação mais organizada.

Tais práticas ajuda no processo de **depuração**.

**Depuração** corresponde ao processo de localizar e corrigir erros ou bugs no código-fonte de qualquer software.

## Hands-Up: Organizando e Documentando Workflows para Manutenção Eficiente

* Adicione uma **Sticky note** com o texto: "## automação falta realizar etapa de enviar por email".
* Utilize **Sticky note** para criar **regions**.
    * Crie uma região com o texto **região triggers**, com a cor mostarda, e agrupe o **chat trigger**.
    * Crie uma região com o texto **região: etapa de edição do json base**, com a cor verde, e agrupe os dois **Edit Field**.
* Acesse o menu **...** na bara superior e clique em **Download**. Este arquivo contém as informações da autmação no formato **JSON**.
* Acesse o menu **...** na bara superior e clique em **Import from file..** para importar uma automação.
