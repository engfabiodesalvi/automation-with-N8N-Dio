# Instalação do N8N

Nesta seção é apresentada as instalações do pacote com o código fonte do **N8N** e do ambiente **Node.js**.\
O **Node.js** é utilizado para a execução de código **JavaScript** em todos os lugares, sem a necessidade do navegador e será utilizado para acessar o pacote de arquivos com os códigos fonte do **N8N**.

## Instalação do Node.js

Utilize o site oficial para instalar o **Node.js**.

* [**Site oficial Node.js**](https://nodejs.org/en)

### Verificando a versão da instalação

* **Node.js:** `node -v`
* **npm:** `npm -v`

## Instalação do N8N

* Utilizando o **npm** para instalar o **N8N**:

    `npm install n8n -g`

* Iniciando o **N8N**:

    `n8n start`

* O editor esta acessível via:

    `http://localhost:5678`
