# Dicas e soluções de problemas

Alguns problemas de instalação podem ocorrer durante a insalação do **n8n**.

* Instalação utilizando o **Node.js**
    * Ficar atento a versão instalada: verificar sempre se a versão instalada é a última versão estável:
        *  Utilize o site do [**npm**](https://www.npmjs.com/) e procure pelas versões estáveis do [**n8n**](https://www.npmjs.com/package/n8n?activeTab=versions)

* Instalação em **Docker** ou **Cloud**
    * Utilize a guia **Logs** para verificar as mensagens reportadas durante a instalação. 
        * Chaves de criptografia utilizadas
        * Porta utilizada pelo **n8n**
        * Dados de **"Migrations"**
            * "Mudanças na estrutura do banco de dados que o n8n utiliza para armazenar seus dados, como fluxos de trabalho, credenciais, execuções e outras conficurações" - (Felipe na Dio Bootcamp)
        
* Utilizar a plataforma oficial **Cloud** do **N8N** para poder usufluir de todos os recurços e suporte.
* O **N8N** pode ser utilizado em um **VPS**:
    * "**VPS(Virtual Private Server)** é um ambiente de hospedagem que se situa entre a hospedagem compartilhada (onde seu site compartilha recursos com muitos outros sites em um único servidor físico), e um servidor dedicado (onde você tem um servidor físico inteiro para si)." - (Felipe na Dio Bootcamp)
