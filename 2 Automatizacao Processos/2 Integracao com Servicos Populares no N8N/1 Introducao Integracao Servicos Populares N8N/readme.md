# Integração com Serviços Populares no N8N

Introdução à integração com serviços populares

## Objetivo

Integração do N8N com serviços populares:

* **Telegram**
* **Whatsapp**
* **Gogle Sheets**

Primeiro, vamos entender o conceito de integração e por que ela é tão importante para a automação.

Em seguida, vamos mergulhar na prática e configurar conexões com serviços populares.

E então, vamos criar um workflow que combina múltiplos serviços, ampliando muito as possibilidades do N8N.

Por fim, vamos falar sobre como resolver problemas comuns que podem ocorrer no dia a dia com integrações
