# Integração com Serviços Populares no N8N
Como funcionam as integrações no N8N

## Integração entre os sistemas e API Rest

* API Rest é uma arquitetura que utiliza maneiras flexíveis de integrar aplicações por meio de transferência de estado representacional (REST)

As informações são enviadas e recebidas principalmente nos formatos **XML** e **JSON**.

* **API** (Application Programming Interface - Interface de programação de aplicações)

    * Extensible Markup Language (XML) é uma linguagem de marcação usada na formatação de documentos para que eles sejam lidos e interpretados por diferentes sistemas.

    * JavaScript Object Ntation (JSON) é um frmato de texto leve usado para armazenar e trocar dados de forma legível por humanos e analisável por máquinas.

* **Integrações no N8N**
    * **Nativas:** Estão implementadas no ambiente, possuindo um bloco (nó) que pode ser configurado.
    * **Não-nativas:** Dependem do envio de requisições Rest para outro ambiente para funcionarem.
